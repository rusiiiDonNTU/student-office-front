# Кабінет студента - код клієнтської частини

Це фронт-енд частина "Кабінету студента", розроблена на React.

## Як встановити 

- Завантажуєте файли (`git clone` або власноруч зі сторінки).
- Відкриваєте папку `front-end` через термінал.
- Пишите `npm install` (повинен бути встановлений Node.js).

## Як збілдати

- В консолі, в директорії `front-end` пишите `npm run build`.
- Згенерується папка `dist` - її і потрібно перенести на бек-енд.

## Як запустити девелопмент-сервер (для тестування фронта окремо від беку) 

- В консолі, в директорії `front-end` пишите `npm run dev`.
- Переходите за адресою в консолі (`http://localhost:5173/`).
