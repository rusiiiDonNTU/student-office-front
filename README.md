# Кабінет студента - код клієнтської частини

Це фронт-енд частина "Кабінету студента", розроблена на React.

## Як запустити

- Завантажуєте файли (`git clone` або власноруч зі сторінки).
- Відкриваєте папку `front-end` через термінал.
- Пишите `npm install` (повинен бути встановлений Node.js).
- Окремо пишете наступні команди установок:
    - `npm install react-route-dom`.
- Запускаєте dev-сервер через команду `npm run dev`.
- Переходите за адресою, що з'явилась в терміналі (`http://localhost:5173/`).
